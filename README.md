## 📖 Descripción del Proyecto

**Happiness & Co** es un proyecto de de primero de DAW **1º de DAW (Desarrollo de Aplicaciones Web)**. La pagina web es para una agenda cultural, festivales, obras de teatro y exposiciones. El proyecto entrelaza de manera técnica una moderna interfaz web con un sólido núcleo de lógica orientada a objetos y estructuras de datos (Programación en Java).

Con un fuerte enfoque en asimilar las competencias del curso, la plataforma luce una estética.
---

## ✨ Características Principales

### 🎨 Frontend Moderno e Interactivo
- **Diseño Responsive y Fluido:** Totalmente adaptable a dispositivos móviles y escritorio usando estándares de CSS moderno.
- **Identidad Visual Consistente:** Clasificación intuitiva de eventos utilizando los colores de la marca:
  - 🎵 **Música & Festivales**: Rosa Vibrante (`#ba007c`)
  - 🎭 **Teatro & Cine**: Azul Profundo (`#009bdb`)
  - 🎪 **Exposiciones & Ferias**: Naranja Cálido (`#ff7b00`)
- **Arquitectura de Vistas Completa:** 
  - 🏠 **Inicio:** Dashboard principal con eventos destacados y un moderno buscador.
  - 📅 **Próximos Eventos:** Catálogo completo en formato lista minimalista.
  - 🕰️ **Historial:** Archivo de eventos pasados con sus respectivas galerías fotográficas.
  - 🙋 **Sobre Nosotros & Contacto:** Páginas informativas con formularios interactivos animado.
  - 📄 **Casi 20 páginas de eventos individuales**, todas dotadas de secciones de información rápida (fechas, ubicaciones, precios).

### 📡 Integración RSS Automática
- Configuración de un **feed XML válido (`eventos.xml`)** directamente integrado, que expone de forma sindicada nuestro gran catálogo de eventos futuros, preparado para integrarse con lectores RSS externos.

### ☕ Núcleo Lógico en Java (Backend)
Robustez en la gestión de datos orientada a objetos:
- **Entidades Clave:** Modeado meticuloso de clases fundamentales: `Usuario`, `Evento`, `Galeria`, `Favorito`.
- **Estructuras de Datos Eficientes:** Implementación intensiva y óptima usando **`HashMap`** y **`ArrayList`** para las relaciones de 1 a N (ej. Evento a Galerías).
- **Gestor Interactivo CLI:** Una aplicación main `do-while` de consola con **9 funciones CRUD completas** para la administración administrativa del sistema.

## 🚀 Cómo Empezar

1. Clona este repositorio o descarga el ZIP.
2. Abre el archivo `index.html` directamente en tu navegador web de preferencia para experimentar la web. Automáticamente enlazará de manera local a todas las partes del proyecto.
3. El fichero `eventos.xml` está en la raíz, listo para ser consumido.
4. Para la parte de consola, carga las clases en cualquier entorno de desarrollo Java (IntelliJ, Eclipse o VS Code) y compila el archivo `Main.java`.
